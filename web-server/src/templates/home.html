Bem-vindo à página de meetups de Rust em Curitiba! Aqui você encontrará informações sobre o grupo e a linguagem de programação Rust.

Sobre o Grupo

Somos um grupo de entusiastas da linguagem de programação Rust em Curitiba. Realizamos meetups regulares para compartilhar conhecimento, discutir novas ideias e colaborar em projetos.

Sobre Rust

Rust é uma linguagem de programação que ajuda você a construir software confiável e eficiente. É ótimo para sistemas de baixo nível, mas também é perfeito para web, desenvolvimento de jogos e muito mais!
